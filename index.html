Cost-sensitive-biotxtm

A general framework for Biomedical Text Mining

Download .zip Download .tar.gz View on GitHub

Cost sensitive learning applied to Disease and Trait recognition

This is an implementation of the methodology described in the paper [https://scholar.google.com/citations?view_op=view_citation&hl=en&user=O75HIQ0AAAAJ&citation_for_view=O75HIQ0AAAAJ:qjMakFHDy7sC here].

This repository contains code for the mining of diseases and traits.

Dataset : [https://www.ebi.ac.uk/gwas/docs/downloads GWAS Catalog]
